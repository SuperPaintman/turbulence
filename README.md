# Turbulence - Tab Manager

---

## How to build

Requirement:

- **Node.js** > 12.x

```sh
# Install NPM dependencies.
$ npm install

# Run the build script.
$ npm run build

# Your extension will be in the public/extension dir.
$ cd public/extension
$ ls -lah
```
